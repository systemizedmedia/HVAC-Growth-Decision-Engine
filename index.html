Math-based budget allocation tool

Growth Decision Engine

Choose the timeframe you want to analyze, enter each platform's numbers, and see which channel is most efficient based on actual input data.

Core logic

No arbitrary scores. No weighted formulas. Only actual inputs, ratios, comparisons, and scenario math.

1. Choose Analysis Timeframe

This should match the timeframe you pulled from ServiceTitan, Meta, Google, LSA, or any other source.

2. Add Channels or Campaigns

Add one row per channel. If you want campaign-level detail, use the optional Campaign / Segment column.
